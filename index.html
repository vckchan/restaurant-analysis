About This Project

This project analyzes restaurant reviews in Davis, CA using sentiment analysis and natural language processing techniques. The visualizations show how customer sentiment relates to traditional ratings and helps identify the most consistently positively-reviewed restaurants.

Created by Candice Chan, Vicky Chan, and Daniel Cayapan for STA 141B.
